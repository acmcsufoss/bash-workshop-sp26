# Redirections excersise

This one will be a light one. 

There is a log.txt file. In this log.txt file you must write when a user logs in and logs off. Due to poor managment, this log file must be wiped clear at the start of a new day.

Write a shell script (multiple is fine) to accomplish this task.

Example log.txt:
```
[   SERVER  ]   This is the start of the log.
[   LOGIN   ]   Bob has logged on at Thu Apr 16 09:19:01 PDT 2026 
[   LOGIN   ]   Kelly has logged on at Thu Apr 16 09:19:12 PDT 2026
[   LOGOUT  ]   Kelly has logged out at Thu Apr 16 17:30:05 PDT 2026
[   LOGOUT  ]   Bob has logged out at Thu Apr 16 18:00:32 PDT 2026
```

Note: There is a UNIX command `date` you can use to grab the current time
