# Pipes Exercise

You may have noticed a file called `beemovie.txt`.

Yes, that is the Bee Movie script, Kelly at work is curious about how the Bee Movie script is structured. So one night, you suddenly find the Bee Movie script on your computer. 

Let’s use some tools to investigate the script!

---

##### Goals

Try to complete the following using pipes (`|`) 

1. Count how many lines are in `beemovie.txt`
2. Count how many lines contain the word `"Bee"`
3. Count how many lines contain the word `"bee"` (lowercase)
4. Find how many words are in the script


##### Hints

You will likely use `cat`, `wc`, and `grep`

Remember:

* `wc -l`: counts lines
* `wc -w`: counts words

`grep` searches for patterns in text
Try chaining multiple commands together


##### Challenge

Can you do all of this without using `cat`?


##### Bonus

* What’s the difference between searching `"Bee"` vs `"bee"`?
* Can you make `grep` ignore case?

Have fun exploring!
